Capabilities

Real tools. Transparent reasoning.

Unlike a plain chatbot, Zenova can call tools when it needs fresh information, then shows you exactly what it did. Every answer comes with its receipts.

🌐

Live Web Search

Pulls current information for CVEs, recent releases, error messages, and documentation. No stale answers from a frozen knowledge cutoff.

🔍

Static Code Analysis

Parses Python with AST and flags common issues — mutable defaults, bare excepts, eval, shell injection risks. No code execution, fully safe.

📘

RFC Lookup

Built-in knowledge of foundational networking RFCs — TCP, BGP, OSPF, HTTP, IPv4/IPv6, DHCP, ARP — with direct links to the source documents.

📎

File-Aware

Attach Python, JS, JSON, YAML, or log files directly. Zenova reads them, includes the contents in context, and answers questions about your code.

💭

Visible Reasoning

Every tool call and result appears inline as a collapsible "Thinking" block. You see what the agent did before reading its answer.

🔓

Open Source, Private

Built on Qwen2.5-Coder-32B via Hugging Face Inference Providers. No proprietary lock-in. The full source is on GitHub.

Under the hood

A ReAct agent on open-source LLMs.

Zenova uses a tool-calling agent loop: the model decides when to invoke a tool, the host executes it, and the result feeds back into the next step — up to four rounds per question. Built without a single proprietary API.

Backbone Model

Qwen2.5-Coder-32B-Instruct via Hugging Face Inference Providers — strong on code, networking, and reasoning.

Interface

Gradio 5 with custom theming and streamed responses. Deployed as a free Hugging Face Space.

Search

DuckDuckGo via duckduckgo-search — no API key, no tracking.

Agent Loop

Custom ReAct-style implementation in Python. Tool calls emitted as parsed JSON, results fed back, capped at four steps.
